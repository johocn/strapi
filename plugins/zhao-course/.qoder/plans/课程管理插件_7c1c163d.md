# Strapi v5 课程管理插件开发计划

## 插件名称: zhao-course

## 数据模型设计

### 1. 课程分类 (course-category)
| 字段 | 类型 | 说明 |
|------|------|------|
| name | string(必填) | 分类名称 |
| description | text | 分类描述 |
| sort | integer | 排序值 |

### 2. 课程标签 (course-tag)
| 字段 | 类型 | 说明 |
|------|------|------|
| name | string(必填) | 标签名称 |
| color | string | 标签颜色 |

### 3. 课程 (course)
| 字段 | 类型 | 说明 |
|------|------|------|
| title | string(必填) | 课程标题 |
| description | text | 课程描述 |
| cover | media(单图) | 封面图 |
| category | relation(多对一) | 所属分类 → course-category |
| tags | relation(多对多) | 课程标签 → course-tag |
| sort | integer | 排序值 |
| publishedAt | datetime | 发布时间 |
| enablePoints | boolean | 是否启用积分（默认false） |
| points | integer | 课程积分值 |
| pointsType | enumeration(course_points,lesson_points) | 积分获得方式：a.课程内积分设置 b.课时内积分设置 |
| isPaid | boolean | 是否收费（默认false） |
| price | decimal(10,2) | 价格（isPaid为true时有效） |

### 4. 知识点 (knowledge-point)
| 字段 | 类型 | 说明 |
|------|------|------|
| title | string(必填) | 知识点标题 |
| content | richtext | 知识点内容 |
| course | relation(多对一) | 所属课程 → course |
| sort | integer | 排序值 |

### 5. 课时 (course-lesson)
| 字段 | 类型 | 说明 |
|------|------|------|
| title | string(必填) | 课时标题 |
| summary | text | 课时简介 |
| content | richtext | 课时富文本内容 |
| video_url | string | 视频链接 |
| audio_url | string | 音频链接 |
| images | media(多图) | 图片集 |
| attachments | media(多文件) | 附件下载 |
| duration | integer | 时长(秒) |
| course | relation(多对一) | 所属课程 → course |
| sort | integer | 排序值 |
| enablePoints | boolean | 是否启用积分（默认false） |
| points | integer | 课时积分值 |
| pointsType | enumeration(lesson_points,quiz_points) | 积分获得方式：a.课时积分 b.通过问卷答题获得 |

### 6. 用户课程授权 (user-course-auth)
| 字段 | 类型 | 说明 |
|------|------|------|
| user | relation(多对一) | 用户 → plugin::users-permissions.user |
| course | relation(多对一) | 课程 → course |
| authType | enumeration(free,paid,admin_grant) | 授权方式：免费/付费/管理员授权 |
| expiresAt | datetime | 授权过期时间（null=永不过期） |
| isExpired | boolean | 是否已过期 |

### 7. 课程学习记录 (course-progress)
| 字段 | 类型 | 说明 |
|------|------|------|
| user | relation(多对一) | 用户 → plugin::users-permissions.user |
| course | relation(多对一) | 课程 → course |
| completedLessons | integer | 已完成课时数 |
| totalLessons | integer | 总课时数 |
| progress | decimal(5,2) | 学习进度百分比(0-100) |
| isCompleted | boolean | 是否完成课程 |
| pointsEarned | integer | 已获得积分 |
| isPointsClaimed | boolean | 是否已领取积分 |
| lastStudyAt | datetime | 最后学习时间 |

### 8. 课时学习记录 (lesson-progress)
| 字段 | 类型 | 说明 |
|------|------|------|
| user | relation(多对一) | 用户 → plugin::users-permissions.user |
| lesson | relation(多对一) | 课时 → course-lesson |
| course | relation(多对一) | 课程 → course |
| progress | decimal(5,2) | 学习进度百分比(0-100) |
| playPosition | integer | 播放位置(秒) |
| duration | integer | 总时长(秒) |
| isCompleted | boolean | 是否完成 |
| isAnswered | boolean | 是否已答题 |
| isCorrect | boolean | 答题是否正确 |
| pointsEarned | integer | 已获得积分 |
| isPointsClaimed | boolean | 是否已领取积分 |
| lastStudyAt | datetime | 最后学习时间 |

## 关联关系
- course-category ← course: 多对一（多个课程属于一个分类）
- course ↔ course-tag: 多对多（课程可以有多个标签）
- course ← knowledge-point: 多对一（多个知识点属于一个课程）
- course ← course-lesson: 多对一（多个课时属于一个课程）
- user ← user-course-auth: 多对一（用户对课程的授权记录）
- course ← user-course-auth: 多对一（课程下的授权记录）
- user ← course-progress: 多对一（用户的课程学习进度）
- course ← course-progress: 多对一（课程下的学习进度记录）
- user ← lesson-progress: 多对一（用户的课时学习进度）
- course-lesson ← lesson-progress: 多对一（课时下的学习进度记录）
- course ← lesson-progress: 多对一（课程下的课时进度记录）

## 依赖插件

### zhao-auth（认证授权，位于 e:\code\plugins\zhao-auth）
提供 JWT 认证和策略授权能力，在路由中间件中使用：

- 认证中间件: plugin::zhao-auth.authenticate
- 授权中间件: { name: "plugin::zhao-auth.authorize", config: { policies: [...] } }
- 内置策略: is-authenticated, has-role, has-channel-access
- 服务: strapi.plugin('zhao-auth').service('auth') 提供 authenticate/authorize/extractToken/getUser
- 类型: AuthUser, PolicyConfig, PolicyResult, AuthService 等

### zhao-common（通用基础设施，位于 e:\code\plugins\zhao-common）
提供日志、错误处理、国际化、配置管理：

- logger: strapi.plugin('zhao-common').service('logger') - info/warn/error/debug
- error-handler: strapi.plugin('zhao-common').service('error-handler') - createError/wrapError/formatError
- i18n: strapi.plugin('zhao-common').service('i18n') - t(code, params)/setMessages(messages)
- config-manager: strapi.plugin('zhao-common').service('config-manager') - get(key, default)/getAll()
- 异常类: AppError, NotFoundError, ForbiddenError, UnauthorizedError, ValidationError
- 错误码: ErrorCodes (COMMON_001~007, CHANNEL_001~007, AUTH_001~005)

### 课程相关错误码和国际化（通过 i18n.setMessages 注册到 zhao-common）

## 实现任务

### Task 1: 初始化插件基础结构
- 创建 package.json（含 strapi 插件配置，声明对 zhao-auth 和 zhao-common 的依赖）
- 创建 TypeScript 配置文件
- 创建 .gitignore

### Task 2: 服务器端入口与生命周期
- server/src/index.ts - 导出所有模块
- server/src/register.ts - 注册课程相关 i18n 消息到 zhao-common
- server/src/bootstrap.ts
- server/src/destroy.ts
- server/src/config/index.ts

### Task 3: 内容类型 Schema 定义
- 8个 schema.json 文件（含新增的 user-course-auth、course-progress、lesson-progress）
- content-types/index.ts 聚合导出

### Task 4: Service 层 - CRUD 业务逻辑
- 8个 service 文件，每个实现 find/findOne/create/update/delete
- 使用 zhao-common 的 logger 记录关键操作日志
- 使用 zhao-common 的 error-handler 创建规范异常
- 使用 zhao-common 的 i18n 获取本地化错误消息
- course-progress service: 更新进度、计算完成状态、领取积分（校验课程积分规则）
- lesson-progress service: 更新播放位置、答题记录、领取积分（校验课时积分规则）
- user-course-auth service: 检查授权、授权过期验证、免费课程自动授权
- services/index.ts 聚合导出

### Task 5: Controller 层 - HTTP 请求处理
- 8个 controller 文件，调用对应 service
- 统一异常处理：catch 中使用 error-handler.wrapError 和 formatError
- lesson-progress controller: 上报进度、答题、领取积分
- course-progress controller: 查询我的进度、领取课程积分
- user-course-auth controller: 检查授权状态
- controllers/index.ts 聚合导出

### Task 6: 路由定义
- admin 路由：8个内容类型的 CRUD，使用 Strapi 默认 admin 认证
- content-api 路由：
  - 公开查询接口：无中间件
  - 用户认证接口：使用 plugin::zhao-auth.authenticate + is-authenticated 策略
  - 参照 zhao-channel 的路由中间件模式编写
- routes/index.ts 聚合导出

### Task 7: Admin 前端
- pluginId.ts
- index.ts（注册插件菜单）
- PluginIcon.tsx
- Initializer.tsx
- App.tsx 和 HomePage.tsx
- 翻译文件（en.json / zh-Hans.json）

### Task 8: 编译验证
- 安装依赖
- 编译 TypeScript
- 验证无错误

### 路由中间件示例（参照 zhao-channel 模式）
```typescript
// 需要用户认证的路由
{
  method: 'POST',
  path: '/my/lesson-progress',
  handler: 'lesson-progress.reportProgress',
  config: {
    middlewares: [
      'plugin::zhao-auth.authenticate',
      {
        name: 'plugin::zhao-auth.authorize',
        config: { policies: [{ name: 'is-authenticated' }] },
      },
    ],
  },
}

// 公开路由（无中间件）
{
  method: 'GET',
  path: '/courses',
  handler: 'course.find',
}
```
