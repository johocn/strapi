# zhao-studio 使用手册

## 安装

```bash
npm install zhao-studio
```

## 配置

在 `config/plugins.ts` 中配置：

```typescript
export default ({ env }) => ({
  'zhao-studio': {
    enabled: true,
    resolve: './plugins/zhao-studio',
  },
});
```

## 功能使用

### 1. 配置采集源

1. 进入 Strapi Admin
2. 点击 **zhao-studio** 插件
3. 进入 **采集源管理**
4. 创建采集源（预设模板或自定义）

### 2. 配置发布平台

1. 进入 **发布平台管理**
2. 创建发布平台类型
3. 创建发布账号（一个平台可配置多个账号）

### 3. 发布文章

1. 进入 **草稿文章管理**
2. 选择文章
3. 选择发布账号
4. 点击发布
