# zhao-oss 对象存储插件手册

## 概述

zhao-oss 是基于 Strapi v5 的对象存储插件，提供文件备份、云存储同步和多云服务商支持。

**核心功能**:
- 多云存储提供商支持（阿里云 OSS 等）
- 文件自动备份到云端
- 批量同步已有文件
- 同步状态管理与统计
- 提供者注册中心

## 安装与配置

### 1. 安装插件

```bash
npm install zhao-oss
```

### 2. 启用插件

在 `config/plugins.ts` 中配置：

```typescript
export default ({ env }) => ({
  "zhao-oss": {
    enabled: true,
    resolve: "./node_modules/zhao-oss",
    config: {
      providers: [
        {
          name: "aliyun",
          enabled: true,
          primary: true,
          options: {
            accessKeyId: env("ALIYUN_ACCESS_KEY"),
            accessKeySecret: env("ALIYUN_ACCESS_SECRET"),
            bucket: env("ALIYUN_OSS_BUCKET"),
            region: env("ALIYUN_OSS_REGION"),
          },
        },
      ],
      maxRetries: 3,
      uploadTimeoutMs: 30000,
    },
  },
});
```

## 数据模型

### Sync Record（同步记录）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| fileId | integer | 是 | 关联的上传文件 ID |
| fileHash | string | 否 | 文件 MD5 哈希 |
| status | enum | 否 | 同步状态（pending/syncing/success/failed/deleted） |
| provider | string | 否 | 使用的存储提供者 |
| remoteUrl | string | 否 | 云端文件 URL |
| remoteEtag | string | 否 | 云端文件 ETag |
| retryCount | integer | 否 | 重试次数 |
| errorMessage | text | 否 | 错误信息 |
| lastSyncedAt | datetime | 否 | 最后同步时间 |

## 服务 API

### Provider Registry (`zhao-oss.provider-registry`)

#### getProvider

获取指定提供者实例。

**签名**:
```typescript
getProvider(name: string): OssProvider | undefined
```

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 提供者名称 |

**响应**: `OssProvider | undefined` - 提供者实例

#### getPrimaryProvider

获取当前主提供者。

**签名**:
```typescript
getPrimaryProvider(): OssProvider | undefined
```

**响应**: `OssProvider | undefined` - 主提供者实例

#### isPrimaryHealthy

检查主提供者健康状态。

**签名**:
```typescript
isPrimaryHealthy(): Promise<boolean>
```

**响应**: `boolean` - 健康状态

#### reloadProviders

重新加载所有提供者。

**签名**:
```typescript
reloadProviders(config: PluginConfig): Promise<void>
```

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| config | PluginConfig | 是 | 插件配置 |

#### getActiveProviders

获取当前可用提供者列表。

**签名**:
```typescript
getActiveProviders(): string[]
```

**响应**: `string[]` - 提供者名称列表

#### getProviderTypes

获取所有已注册提供者类型。

**签名**:
```typescript
getProviderTypes(): string[]
```

**响应**: `string[]` - 提供者类型列表

### Sync Service (`zhao-oss.sync-service`)

#### backupFile

上传文件到云端备份。

**签名**:
```typescript
backupFile(fileId: number): Promise<void>
```

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| fileId | number | 是 | 上传文件 ID |

**说明**: 自动重试 3 次，支持指数退避

#### batchSync

批量同步已有文件。

**签名**:
```typescript
batchSync(limit?: number, offset?: number): Promise<{ total: number; success: number; failed: number }>
```

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| limit | number | 否 | 每次同步数量（默认 50） |
| offset | number | 否 | 起始偏移（默认 0） |

**响应**: 同步统计

#### deleteRemote

删除云端文件。

**签名**:
```typescript
deleteRemote(recordId: number): Promise<void>
```

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| recordId | number | 是 | 同步记录 ID |

#### checkSyncStatus

检查文件同步状态。

**签名**:
```typescript
checkSyncStatus(fileId: number): Promise<{ synced: boolean; provider?: string; remoteUrl?: string }>
```

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| fileId | number | 是 | 上传文件 ID |

**响应**: 同步状态信息

#### getSyncStats

获取同步统计。

**签名**:
```typescript
getSyncStats(): Promise<{ total: number; synced: number; failed: number; pending: number }>
```

**响应**: 同步统计信息

export default ({ env }) => ({

## 使用示例

### 示例1：备份单个文件

```typescript
const syncService = strapi.plugin("zhao-oss").service("sync-service");

try {
  await syncService.backupFile(123);
  console.log("文件备份成功");
} catch (error) {
  console.error("文件备份失败:", error.message);
}
```

### 示例2：批量同步

```typescript
const syncService = strapi.plugin("zhao-oss").service("sync-service");

const result = await syncService.batchSync(100, 0);
console.log(`同步完成: ${result.success}/${result.total}`);
```

### 示例3：检查同步状态

```typescript
const syncService = strapi.plugin("zhao-oss").service("sync-service");

const status = await syncService.checkSyncStatus(123);
if (status.synced) {
  console.log("已同步到:", status.remoteUrl);
} else {
  console.log("未同步");
}
```

### 示例4：获取同步统计

```typescript
const syncService = strapi.plugin("zhao-oss").service("sync-service");

const stats = await syncService.getSyncStats();
console.log(`总记录: ${stats.total}`);
console.log(`已同步: ${stats.synced}`);
console.log(`失败: ${stats.failed}`);
console.log(`待同步: ${stats.pending}`);
```

## 注意事项

1. **提供者配置**: 至少需要配置一个启用的提供者
2. **主提供者**: 必须指定一个 primary 提供者
3. **文件大小**: 建议限制单个文件大小（根据云服务商限制）
4. **重试机制**: 上传失败会自动重试 3 次，使用指数退避
5. **哈希校验**: 使用 MD5 哈希验证文件一致性
6. **权限配置**: 确保云存储账号有正确的读写权限
