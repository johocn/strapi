# zhao-quiz 测验插件手册

## 概述

zhao-quiz 是基于 Strapi v5 的测验插件，提供完整的题目管理、测验配置和答题记录功能。

**核心功能**:
- 多种题型支持（单选、多选、判断、填空、简答等）
- 测验配置（时间限制、及格分数、重试次数）
- 答题记录追踪
- 测验成绩统计
- 批量导入题目

## 安装与配置

### 1. 安装插件

```bash
npm install zhao-quiz
```

### 2. 启用插件

在 `config/plugins.ts` 中配置：

```typescript
export default ({ env }) => ({
  "zhao-quiz": {
    enabled: true,
    resolve: "./node_modules/zhao-quiz",
  },
});
```

## 数据模型

### Quiz（题目）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | richtext | 是 | 题目内容 |
| type | enum | 是 | 题型（single_choice/multiple_choice/true_false/fill_blank/short_answer/matching/ordering） |
| options | json | 否 | 选项（选择题使用） |
| answer | text | 否 | 正确答案 |
| explanation | richtext | 否 | 答案解析 |
| difficulty | enum | 否 | 难度（easy/medium/hard），默认 medium |
| points | integer | 否 | 分值，默认 0 |
| sort | integer | 否 | 排序号，默认 0 |
| isPublished | boolean | 否 | 是否发布，默认 false |
| course | relation | 否 | 所属课程 |
| lesson | relation | 否 | 所属章节 |
| knowledgePoints | relation | 否 | 关联知识点 |

### Quiz Record（答题记录）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| user | relation | 是 | 用户 |
| quiz | relation | 是 | 题目 |
| answer | json | 否 | 用户答案 |
| isCorrect | boolean | 否 | 是否正确 |
| score | decimal | 否 | 得分 |
| totalPoints | integer | 否 | 题目总分 |
| submittedAt | datetime | 否 | 提交时间 |
| duration | integer | 否 | 答题时长（秒） |
| course | relation | 否 | 所属课程 |
| lesson | relation | 否 | 所属章节 |

### Quiz Exam（测验配置）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | string | 是 | 测验标题 |
| description | text | 否 | 测验描述 |
| timeLimit | integer | 否 | 时间限制（秒），默认 0（无限制） |
| passScore | decimal | 否 | 及格分数，默认 60 |
| totalPoints | integer | 否 | 总分 |
| questionCount | integer | 否 | 题目数量 |
| randomOrder | boolean | 否 | 随机出题，默认 false |
| allowRetry | boolean | 否 | 允许重试，默认 true |
| maxAttempts | integer | 否 | 最大重试次数，默认 0（无限制） |
| showResult | boolean | 否 | 显示结果，默认 true |
| course | relation | 否 | 所属课程 |
| lesson | relation | 否 | 所属章节 |
| questions | relation | 否 | 关联题目（多对多） |

### Quiz Exam Attempt（考试记录）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| user | relation | 是 | 用户 |
| exam | relation | 是 | 测验配置 |
| answers | json | 否 | 用户答案集合 |
| totalScore | decimal | 否 | 总得分 |
| isPassed | boolean | 否 | 是否通过 |
| startedAt | datetime | 否 | 开始时间 |
| submittedAt | datetime | 否 | 提交时间 |
| duration | integer | 否 | 考试时长（秒） |
| attemptNumber | integer | 否 | 尝试次数，默认 1 |

### Quiz Batch（批量导入）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | media | 是 | 导入文件（Excel/CSV） |
| status | enum | 否 | 导入状态（pending/processing/completed/failed） |
| totalCount | integer | 否 | 总行数 |
| successCount | integer | 否 | 成功导入数 |
| failedCount | integer | 否 | 失败数 |
| errorDetails | json | 否 | 错误详情 |
| createdAt | datetime | 否 | 创建时间 |

## API 端点

### 题目管理

#### 查询题目列表

```
GET /api/zhao-quiz/quiz
```

**Query 参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| page | integer | 页码 |
| pageSize | integer | 每页数量 |
| type | string | 题型 |
| difficulty | string | 难度 |
| course | string | 课程 ID |
| lesson | string | 章节 ID |

#### 创建题目

```
POST /api/zhao-quiz/quiz
```

**请求体**:

```json
{
  "title": "以下哪个是正确的 JavaScript 声明方式？",
  "type": "single_choice",
  "options": [
    { "key": "A", "text": "var x = 1" },
    { "key": "B", "text": "variable x = 1" },
    { "key": "C", "text": "declare x = 1" }
  ],
  "answer": "A",
  "explanation": "var 是 JavaScript 的变量声明关键字",
  "difficulty": "easy",
  "points": 10
}
```

#### 更新题目

```
PUT /api/zhao-quiz/quiz/:id
```

#### 删除题目

```
DELETE /api/zhao-quiz/quiz/:id
```

### 测验管理

#### 创建测验

```
POST /api/zhao-quiz/quiz-exam
```

**请求体**:

```json
{
  "title": "第一章单元测试",
  "description": "测试第一章学习内容",
  "timeLimit": 1800,
  "passScore": 60,
  "totalPoints": 100,
  "questionCount": 10,
  "randomOrder": true,
  "allowRetry": true,
  "maxAttempts": 3,
  "showResult": true,
  "course": "doc_xxx",
  "questions": ["doc_aaa", "doc_bbb", "doc_ccc"]
}
```

#### 开始测验

```
POST /api/zhao-quiz/quiz-exam-attempt
```

**请求体**:

```json
{
  "exam": "doc_xxx",
  "user": 123,
  "startedAt": "2024-01-01T10:00:00Z"
}
```

#### 提交测验答案

```
PUT /api/zhao-quiz/quiz-exam-attempt/:id
```

**请求体**:

```json
{
  "answers": {
    "doc_aaa": "A",
    "doc_bbb": ["A", "C"],
    "doc_ccc": "正确答案"
  },
  "submittedAt": "2024-01-01T10:30:00Z",
  "duration": 1800
}
```

## 服务 API

### Quiz Service (`zhao-quiz.quiz`)

#### find

查询题目列表。

**签名**:
```typescript
find(query?: any): Promise<any[]>
```

**示例**:
```typescript
const quizService = strapi.plugin("zhao-quiz").service("quiz");
const questions = await quizService.find({
  filters: { type: "single_choice", difficulty: "easy" },
});
```

#### findOne

查询单个题目。

**签名**:
```typescript
findOne(documentId: string): Promise<any>
```

#### findByType

按题型查询。

**签名**:
```typescript
findByType(type: string, query?: any): Promise<any[]>
```

#### findByCourse

按课程查询题目。

**签名**:
```typescript
findByCourse(courseDocumentId: string, query?: any): Promise<any[]>
```

#### findByLesson

按章节查询题目。

**签名**:
```typescript
findByLesson(lessonDocumentId: string, query?: any): Promise<any[]>
```

---

### Quiz Exam Service (`zhao-quiz.quiz-exam`)

#### create

创建测验配置。

**签名**:
```typescript
create(data: any): Promise<any>
```

#### startAttempt

开始测验尝试。

**签名**:
```typescript
startAttempt(examDocumentId: string, userId: number): Promise<any>
```

#### submitAttempt

提交测验答案。

**签名**:
```typescript
submitAttempt(attemptId: string, answers: Record<string, any>): Promise<any>
```

## 使用示例

### 示例1：创建选择题

```typescript
const quizService = strapi.plugin("zhao-quiz").service("quiz");

const question = await quizService.create({
  title: "HTTP 默认端口是多少？",
  type: "single_choice",
  options: [
    { "key": "A", "text": "21" },
    { "key": "B", "text": "22" },
    { "key": "C", "text": "80" },
    { "key": "D", "text": "443" }
  ],
  answer: "C",
  explanation: "HTTP 默认端口是 80，HTTPS 默认端口是 443",
  difficulty: "easy",
  points: 5
});
```

### 示例2：创建测验并开始

```typescript
const examService = strapi.plugin("zhao-quiz").service("quiz-exam");
const attemptService = strapi.plugin("zhao-quiz").service("quiz-exam-attempt");

// 创建测验
const exam = await examService.create({
  title: "基础知识测验",
  timeLimit: 300,
  passScore: 60,
  questions: ["doc_aaa", "doc_bbb"]
});

// 开始测验
const attempt = await attemptService.startAttempt(exam.documentId, 123);
```

### 示例3：提交答案并评分

```typescript
const attemptService = strapi.plugin("zhao-quiz").service("quiz-exam-attempt");

const result = await attemptService.submitAttempt(
  "attempt_xxx",
  {
    "doc_aaa": "A",
    "doc_bbb": "正确答案"
  }
);

console.log(`得分: ${result.totalScore}, 是否通过: ${result.isPassed}`);
```

## 注意事项

1. **文档模式**: 使用 Strapi v5 的 Documents 模式
2. **题型支持**: 支持 7 种题型，可扩展
3. **测验配置**: 可设置时间限制、及格分数、重试次数等
4. **随机出题**: randomOrder 为 true 时题目顺序随机
5. **答案格式**: 根据题型不同，答案格式不同（单选为字符串，多选为数组）
